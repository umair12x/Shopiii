# Shopiii - Feature Guide

## Home

- Shop profile overview
- Today's, monthly, and yearly summary cards
- Sync status panel

## Daily Book

- Add entry modal
- Edit and delete entries
- Auto profit calculation
- Pending/Collected payment toggle
- Fixed totals summary

## Products

- Store product names, prices, and notes
- Barcode lookup and barcode assignment
- Camera-based barcode scanning on Android/iOS

## History

- View any previous date
- Read-only records for reconciliation

## Dashboard

- 30-day sales/profit trend
- Yearly monthly profit chart
- Analytics cards and summaries

## Settings

- Update shop name, owner, address, contact
- Review data and sync configuration

## Offline support

The app works fully offline because all core data is stored locally first.

## Cloud backup

Firebase can be enabled later for optional backup and multi-device sync.
